# Install deps
pip install -r requirements.txt

# Generate the default dataset
python main.py

# Generate 5,000 customers using the legacy positional argument
python main.py 5000

# Generate a smaller demo dataset
python main.py --customers 100 --inventory 50 --transactions 1000 --output-dir data/demo

# Generate CSV files only
python main.py --skip-db

How to tweak
What to change               Where
Customer count               --customers or DEFAULT_CUSTOMERS in config.py
Inventory/product count      --inventory or DEFAULT_INVENTORY in config.py
Transaction count            --transactions or DEFAULT_TRANSACTIONS in config.py
Output folder                --output-dir
Seed                         --seed or DEFAULT_SEED in config.py
Columns                      Edit *_COLUMNS in config.py and update build_* functions
